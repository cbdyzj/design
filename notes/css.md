# CSS

## HTML

- 意义优先
- 滥用类（classitis），滥用div（divitus）

## 层叠样式

选择器权重 = 1000 * a + 100 * b + 10 * c + d

1. a取决于是否是行内样式
2. b为ID选择器数量
3. c为类、伪类、属性选择器数量
4. d为元素择器、伪元素选择器数量
5. 样式会继承与覆盖

## 盒模型

2. 边框（border）
3. 内边距（padding）
4. 内容区域（height，width）

- 外边距会叠加
- 块级元素、行内元素、行内块
